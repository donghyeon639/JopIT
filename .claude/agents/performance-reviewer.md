## 검사 항목

### 높은 심각도 (즉시 수정)
- N+1 쿼리: 루프 안에서 반복 DB 조회
- 동기 I/O: 비동기 처리 가능한 구간에서 블로킹 호출
- 대용량 데이터 전체 메모리 로드: 페이지네이션 없이 전체 조회

### 중간 심각도 (빠른 수정 권장)
- 중첩 루프: O(n²) 이상 복잡도
- 반복 계산: 루프 안에서 매번 동일한 값 재계산
- 캐싱 미적용: 동일 쿼리/연산 반복 호출

### 낮은 심각도 (개선 권장)
- 비효율적 자료구조: 탐색에 배열 사용 (객체/Map 대체 가능)
- 불필요한 데이터 복사: 대용량 객체 불필요하게 복제
